Welcome to GameAnalytics Unity Android Plugin

- To test the plugin in your Android App, you can simply use the prepared SampleAndroidDeviceInfoPluginScene from the scene folder, add it to your build scene and run your app, it should display your Android device info plus system memory usage, application memory usage, and average app memory usage.

- If you would like to use the plugin in your own scene, simply add AndroidDeviceInfoPlugin game object from the Prefabs folder, you can then assign any Text elements from your scene to the corresponding fields in the script with the plugin game object.
